# Chat de Socket.IO

Um dia faço um readme mais decente.

## Instalação

```
git clone https://github.com/giordanna/socket-chat.git
cd socket-chat
npm i
```

## Execução

```
redis-server
npm run start

# ou se preferir subir com duas instâncias, se tiver pm2 instalado
pm2 start ecosystem.config.js
```
